3.3 · Payment Required is back

HTTP status
402
Payment Required
Reserved 1991. Unused for 30 years. Revived by AI agents in 2025.
Agent hits a paid endpoint → server returns 402 + price → agent signs a stablecoin payment → retries with payment receipt header → gets the resource. One round trip. No accounts, no API keys, no human in the loop.
x402
Coinbase · Cloudflare · Stripe
HTTP 402 revival. Pay-per-request stablecoin micropayments. Cloudflare uses it for pay-per-crawl.
